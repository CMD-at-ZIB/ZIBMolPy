Module zgf_rerun

source code

What it does

This tool will prepare desolvated trajectories and topologies in order to allow energy reruns for your molecule of interest (MOI). Your node samplings can then be rerun with zgf_mdrun. After the rerun is complete, the rerun energies can be used by zgf_reweight.

How it works

At the command line, type:

       $ zgf_rerun [options]

Options

  • ignore-convergence: 
     help: rerun despite not-converged
