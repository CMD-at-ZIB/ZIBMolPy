Module zgf_grompp

source code

What it does

This is the fourth step of ZIBgridfree.

This tool will run grompp for every node, using the mdp file given to zgf_create_pool. It will fail if grompp produces an error. You should also watch out for any NOTE that grompp may have for you.

The next step is zgf_mdrun, or zgf_submit_job_HLRN, if you are working on HLRN.

How it works

At the command line, type:

       $ zgf_grompp
