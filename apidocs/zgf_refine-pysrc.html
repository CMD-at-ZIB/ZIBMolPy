  5  What it does 
  6  ============ 
  7          B{This is the (optional) sixth step of ZIBgridfree.} 
  8   
  9          This tool scans the node pool for nodes that have not converged, or that have converged by mistake. For every non-converged node, two alternatives exist: 
 10   
 11                  - Refinement: Create new nodes from the node sampling and discard the original node. This is useful if the node sampling has come upon an additional minimum. 
 12                  - Extension: Keep the original node and extend the node sampling. 
 13   
 14          L{zgf_refine} calls L{zgf_create_nodes} in order to add new nodes to the node pool. Therefore, it inherits several options from L{zgf_create_nodes}. For more information, please refer to the L{zgf_create_nodes} documentation.  
 15           
 16          B{The next step is L{zgf_mdrun}, or L{zgf_submit_job_HLRN}, if you are working on HLRN.} 
 17   
 18  How it works

 20          At the command line, type::
