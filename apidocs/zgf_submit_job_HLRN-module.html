What it does

This is the fifth step of ZIBgridfree.

This tool performs the task of zgf_mdrun on HLRN. In addition to the inherited zgf_mdrun options, you can pick several options for the HLRN job script, such as job queue, time limit, and e-mail notification. It is also possible to allocate a larger number of nodes and subdivide these into separate zgf_mdrun processes. Alternatively, you can call this tool several times on the same node pool to submit multiple smaller jobs. For more information, please refer to the zgf_mdrun documentation.

The next step is zgf_refine, if you want to refine or extend nodes where convergence has not been achieved. Otherwise, you can proceed with zgf_reweight.

How it works

At the command line, type:

       $ zgf_submit_job_HLRN [options]
